# Sarambabot

A serverless Discord bot that checks a twitter feed every minute & forwards One Piece updates to a specific discord channel

### Test

```
npm run test:pollTwitterFeed
npm run test:sendTweetsToDiscord
```

### Fix code style

```
npx eslint ./src/** --fix
```
