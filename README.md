# Sarambabot

A serverless application that backups a youtube playlist into the wayback machine

## Running tests

```
npm run test:backupVideos
npm run test:queuePlaylistBackup
```

## Fixing code style

```
npx eslint ./src/** --fix
```

## Deploying

```
npm run deploy:dev
```

## Cleaning up

```
npm run remove:dev
```
